# ErgoSNM Animate

An animation introducing [ErgoSNM keyboard](https://github.com/siderakb/ergo-snm-keyboard).

Powered by [Motion Canvas](https://motioncanvas.io/). Sound from [TTSMaker](https://ttsmaker.com/)

## Usage

```bash
pnpm install
pnpm start
```

Render:

```bash
pnpm render

# or

./render.sh
```
